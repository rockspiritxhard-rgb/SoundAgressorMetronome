Метроном для песен

Кликните на песню для запуска метронома, повторный клик - остановка

Песня: Не выбрана
Метроном остановлен
120 BPM

Инструкция:

• Кликните на песню для запуска метронома

• Повторный клик на ту же песню - остановка метронома

• Клик на другую песню - переключение на неё и запуск её метронома

• При необходимости отрегулируйте темп

• Для песен 1-2, 4-18, 19: метроном звучит восьмыми нотами (акцент на каждой первой восьмой)

• Для песни 3: метроном звучит четвертями (акцент на каждой первой четверти)

• Во всех песнях акцент чередуется: акцент - без акцента - акцент - без акцента
